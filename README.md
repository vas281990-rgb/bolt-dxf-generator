# bolt-dxf-generator

Данный скрипт используется для генерации технических чертежей болтов по ЕСКД.  
скрипт создаёт DXF-чертежи, Python-скрипты и 3D-модели для каждого варианта болта.

---

## Для чего это нужно?

Данный генератор позволяет получить сотни уникальных вариантов без ручного черчения —
с разными размерами, углами фасок и длиной резьбы, что значительно ускоряет и упрощает работу.

---

## Что генерируется

Для каждой комбинации параметров скрипт создаёт 3 файла:

| Файл | Описание |
|------|----------|
| `*1.dxf` | Технический чертёж: вид спереди + вид сбоку |
| `*3.py`  | Читаемый Python-скрипт для воспроизведения этого чертежа |
| `*4.py`  | Минифицированный вариант того же скрипта |

Имена файлов содержат дату и время генерации, например:  
`20260302-143022-1.dxf`

---

## Параметры болтов

Размеры по **ГОСТ 7798** (болты с шестигранной головкой):

| Резьба | Размер под ключ S | Высота головки H |
|--------|-------------------|------------------|
| M20    | 30 мм             | 12.5 мм          |
| M24    | 36 мм             | 15.0 мм          |
| M27    | 41 мм             | 17.0 мм          |
| M30    | 46 мм             | 18.7 мм          |
| M36    | 55 мм             | 22.5 мм          |

Длина болта: от 80 до 160 мм, шаг ~9.7 мм  
Итого: ~45 вариантов за один запуск

Случайные параметры (меняются каждый раз):
- угол фаски головки: 13°–39°
- длина резьбовой части: 40–70% от длины болта
- размер фаски торца: 1.0–3.0 мм

---

## Структура чертежа

Каждый DXF содержит два вида:

**Вид спереди** — контур болта с:
- шестигранной головкой и фаской
- цилиндрическим стержнем
- условным обозначением резьбы (по ЕСКД)
- фаской 45° на торце
- размерными линиями

**Вид сбоку** — шестигранная головка с:
- вписанной и описанной окружностями
- тремя диагоналями
- осевыми линиями
- размером под ключ

Слои по ЕСКД:

| Слой | Цвет | Назначение |
|------|------|------------|
| Main | белый | Основные линии контура |
| Thread | голубой | Линии резьбы |
| Axises | пурпурный | Осевые линии |
| Hatches | серый | Штриховка |
| Measure | синий | Размерные линии |

## Установка

Требуется Python 3.9+
```bash
pip install -r requirements.txt
```

Основные зависимости:

| Библиотека | Версия | Зачем |
|------------|--------|-------|
| `ezdxf` | ≥ 1.0.0 | Создание DXF-файлов |
| `python-minifier` | ≥ 2.9.0 | Минификация Python-кода |
| `cadquery` | ≥ 2.3.0 | Генерация 3D-моделей STEP |

> **Примечание:** cadquery устанавливает крупные зависимости (~170 МБ).  

---

## Запуск
```bash
python3 bolt-dxf-generate-synthetic.py
```

Все файлы автоматически сохраняются в папку `output_bolts/`  
В терминале будет выводиться прогресс:
```
DXF saved: D=20, L=80
DXF saved: D=24, L=80
...
```
